日志查看

当前版本不再读取本地日志文件，统一改为查看平台或终端输出。

线上环境

查看 Vercel Runtime Logs

线上部署时，应用日志直接输出到标准输出，推荐在 Vercel 项目面板或 `vercel logs` 中查看。 本页不再尝试读取函数文件系统中的日志文件，以避免无状态环境下的误导信息。

{% if is_vercel %} 当前实例识别为 Vercel 环境，建议直接到项目控制台查看 Runtime Logs。 {% else %} 当前不是 Vercel 环境；如果你在本地调试，请直接查看启动终端输出。 {% endif %}

本地开发

查看启动终端输出

本地运行 `uv run python main.py` 时，所有日志都会直接输出到终端。 如果开启 `DEBUG_LOGGING` 且当前不是 Vercel 环境，仍会额外写入 `logs/` 目录。

运行时日志页已降级为说明页；这是为了兼容 Serverless 的只读文件系统和多实例模型。
